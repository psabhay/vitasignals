Agreed on 3-tab redesign: Home, Metrics, Profile.

**Why:** Current 4-tab layout is tool-oriented (Data/Charts/Reports), not task-oriented. Users think in terms of their health metrics, not app tools.

**How to apply:**
- Home: smart summary, nudges, quick log, goals, recent activity
- Metrics: merged Data+Charts, metric-centric browsing, custom comparison charts, saved views with QUICK access
- Profile: user info, custom metric management, report generation (moved from tab), subscription, data management
- MetricDetailView becomes the unified hub: chart + data points + add reading
- Reports is NOT a tab — it's a section in Profile
